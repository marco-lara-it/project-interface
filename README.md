# project-interface
In this project users can interface a GoogleSheet through a GUI.

- script.py is linked to the document

- script1.py looks for the document on the google drive and after that writes and read 

- script2.py is the adjustment of the guide at this link https://developers.google.com/sheets/api/quickstart/python?hl=it

- managing_app is the version of a smart app connected to a google sheet document with log-in page, create credential page, home page, balance page, clients page and a indicators page.
